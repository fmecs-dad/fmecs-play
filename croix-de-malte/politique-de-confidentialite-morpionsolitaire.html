Politique de Confidentialité

Dernière mise à jour : 27 mars 2026

1. Données collectées

Nous collectons uniquement :

  • Votre pseudo (choisi par vous et affiché publiquement dans les classements).
  • Votre adresse e-mail (uniquement si vous la fournissez pour modifier votre profil ou récupérer votre compte).

L'adresse e-mail n'est pas obligatoire pour jouer ou enregistrer vos scores.

2. Utilisation des données

Vos données servent uniquement à :

  • Afficher vos scores dans le tableau des meilleurs joueurs (via votre pseudo).
  • Vous permettre de modifier votre profil (y compris votre adresse e-mail, si fournie).
  • Vous permettre de supprimer vos données.

3. Durée de conservation

Vos données (pseudo, scores et, le cas échéant, adresse e-mail) sont conservées pendant 1 an après votre dernier enregistrement de score, puis supprimées automatiquement.

4. Vos droits

Conformément au RGPD, vous pouvez :

  • Accéder à vos données (pseudo, scores et adresse e-mail, si fournie).
  • Les modifier ou les supprimer en nous contactant à contact@fmecs.fr.
  • Vous désinscrire via le lien présent dans votre espace joueur.

5. Sécurité

Vos données sont stockées sur les serveurs sécurisés de Supabase, hébergés en Europe. Aucune donnée n’est partagée avec des tiers.

6. Contact

Pour toute question concernant cette politique de confidentialité, contactez-nous à : contact@fmecs.fr.
